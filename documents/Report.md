<h1 align="center"> Отчётные документы по итогам тестирования.</h1>

*  **Описание**

    *  Было проведено автоматизированное тестирование комплексного сервиса покупки тура, взаимодействующего с СУБД и API Банка.

*  **В ходе чего было автоматизированно 34 тест-кейса**.

    *  **25** успешно пройденых тест-кейсов.

    *  **9** тест-кейсов не прошли проверку, подробности в Issues.

    *  В процентном соотношении **73.52%** проиденых и **26.47%** не прошли.
* **Общие рекомендации**
* 
  * Исправить баги в валидации: выявлены ошибки при вводе некорректных данных (пустые поля, спецсимволы), которые пропускаются системой.
  * Добавить ID элементам: для повышения стабильности автотестов рекомендуется добавить атрибуты data-test-id в HTML-разметку.
