<h1 align="center"> Отчётные документы по итогам тестирования.</h1>

*  **Описание**

    *  Было проведено автоматизированное тестирование комплексного сервиса покупки тура.

*  **В ходе чего было автоматизированно 60 тест-кейса**.

    *  **48** успешно пройденых тест-кейсов.

    *  **12** тест-кейсов не прошли проверку, подробности в Issues.

    *  В процентном соотношении **80%** проиденых и **20%** не прошли.
* **Общие рекомендации**
* 
  * Исправить баги в валидации: выявлены ошибки при вводе некорректных данных (пустые поля, спецсимволы), которые пропускаются системой.
  * Добавить ID элементам: для повышения стабильности автотестов рекомендуется добавить атрибуты data-test-id в HTML-разметку.
